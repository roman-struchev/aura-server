AuraPad

A local-first desktop code editor. Everything — dictation, translation, git, terminal — runs on your machine; nothing is required to leave it.

View on GitHub
This server (aura.struchev.site) hosts the Work Together backend — the collaborative-editing relay AuraPad talks to when you share a file: it mints time-limited links and serves the live editor page a recipient opens in their browser, with no AuraPad install required. See docs/edit-together in the repo for how it works.
