AuraPad

A code editor that stays on your machine.

Dictation, translation, git, and a terminal — all built in. Nothing leaves your computer, unless you choose to share a file live with someone else.

It's free and open source — the code is on GitHub: editor and server .

This is also where AuraPad's Work Together feature lives: when you share a file, this is what creates the link and the page the other person opens — no install needed on their end. There's more on how it works in docs/edit-together.
